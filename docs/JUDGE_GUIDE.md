# Judge Guide

## Product Claim

Family Sidekicks turns one shared family memory into a crew of focused AI capabilities. Each Sidekick has a dedicated Corner, a structured tool and an ongoing conversation.

## Three-Minute Path

1. **Family today:** start with the attention queue, demo forecast, timeline, meals, nearby idea and next seven days.
2. **Shared coordination:** complete the field-trip task, switch parents and show that completion remains while the personal Home crew changes.
3. **Personal Home:** use `Choose three` to replace one shortcut for the active parent only.
4. **Family profile:** edit one child name, interest and allergy. Save, reload and show that the dashboard updates and persists.
5. **Adventure Corner:** search the configured city, show sources and save one result.
6. **Kitchen Corner:** create the 25-minute meal. Show the edited allergy as a hard rule.
7. **Talk to Nori:** ask a follow-up without re-entering names, dislikes or allergy information. Show `Memory used`.
8. **Story Corner:** create a story from the edited names and interests, then ask Lumi to change the mood.
9. **Atlas:** upload a worksheet screenshot, show same-level tasks and the printable worksheet plus optional answer key.
10. **Pippa:** add one friend with parent contact details, create the project, then open the recipient-specific WhatsApp/email draft and the real `.ics` blocker.
11. **Quinn:** add the players, start a quiz and show how the active turn and score rotate from one family member to the next.
12. **Romy:** build a trip, ask for a packing change in chat and show `Workbench: Current trip plan`.
13. **Romy connectors:** show CHECK24, Airbnb and hotel-search as clearly labeled simulated MCP handoffs.
14. **Moxie:** create a narrow custom Buddy with explicit memory access.
15. **Judge View:** close on live capabilities, simulations and safety boundaries.

## Result Labels

- `Live AI`: an OpenAI route executed with the server-side project key.
- `Sample result`: a visible deterministic fixture.
- `Real action`: geolocation, print, share draft, calendar export, document metadata upload or download.
- `Simulated`: a commercial or connector preview with no external side effect.
- `Bounded preview`: a live conversation whose professional or source limits remain explicit.

## Fresh-Browser Checks

- Crew Home appears without login.
- Home task completion is shared while the three Home shortcuts remain personal to each parent.
- Family Memory can be edited, exported, imported and survives a reload.
- Featured and extended Corners are reachable on desktop and mobile.
- Conversation history remains separate per Sidekick.
- A generated Corner result appears as the connected Workbench Memory in its chat and remains after reload.
- Every answer can show the family facts it used.
- Manual city works when geolocation is declined.
- Sample and live results are unmistakable.
- Reset clears conversations and saved outputs.
